sshd

Mettre en place un serveur ssh consiste :

  1. à installation les packages nécessaires

  2. à démarrer le service

  3. à adapter le cas échéant les règles du firewall

1. Installer les packages

sur Redhat
# dnf install openssh-server
sur Debian
# apt-get upgrade
# apt-get install openssh-server

2. Démarrer le service

# systemctl start sshd

Pour que le démarrage soit effectif à chaque reboot :

# systemctl enable sshd

3. Adapter le firewall

Si un firewall tourne sur le serveur, alors il est nécessaire d’ajuster les règles de filtrage pour autoriser les clients à se connecter.

Il se peut que l’installation du package adapte automatiquement les règles du firewall. Sinon, pour tester l’exportation sans se poser de question, on peut (exceptionnellement) arrêter le firewall. Sur Redhat, on arrête le firewall par :

# systemctl stop firewalld

Pour le désactiver définitivement (déconseillé) :

# systemctl disable firewalld
